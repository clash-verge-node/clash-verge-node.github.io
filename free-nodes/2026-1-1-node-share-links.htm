重庆科学上网的常用方式

1. VPN（虚拟专用网络）

优点：
- 连接稳定，速度较快
- 提供全面的加密保护
- 支持多设备同时使用

适用场景：
- 需要稳定访问国际网站（如远程办公、在线会议）
- 对隐私保护要求较高的用户

使用方法：
1. 选择可靠的VPN服务商（如ExpressVPN、NordVPN等）。
2. 下载并安装官方客户端。
3. 登录账户，选择目标服务器（如美国、日本等）。
4. 连接成功后即可自由访问全球网络。

2. Shadowsocks（SS）

优点：
- 轻量级，占用资源少
- 适合访问被屏蔽的网站
- 配置灵活，可自建服务器

适用场景：
- 需要高效访问特定网站（如Google、Twitter）
- 对速度要求较高的用户

使用方法：
1. 下载Shadowsocks客户端（如Windows版、Android版）。
2. 获取服务器配置信息（地址、端口、密码等）。
3. 在客户端填入参数并连接。
4. 测试是否能正常访问目标网站。

3. V2Ray（Vmess协议）

优点：
- 抗封锁能力强
- 支持多种传输协议（如WebSocket、TCP）
- 适合高级用户自定义配置

适用场景：
- 网络环境复杂（如企业网络、校园网）
- 需要更高匿名性的用户

使用方法：
1. 下载V2Ray客户端（如V2RayN、Qv2ray）。
2. 导入服务器配置文件（通常为JSON格式）。
3. 启动代理并测试连接。

FAQ

Matsuri 项目是什么？
Matsuri 是基于 sing-box 的跨平台 GUI 客户端，界面现代，支持多种协议和分流方式。它提供直观的节点管理和日志监控功能，适合喜欢图形化操作的用户。Matsuri 的设计目标是简洁、高效、易于上手。
VLESS + XTLS 模式在移动网络下表现如何？
XTLS 模式减少握手次数和加密开销，适合高延迟或不稳定移动网络。结合策略组和多出口节点，可提高访问速度和连接稳定性，保证用户体验流畅。
Matsuri 如何提升节点连接效率？
Matsuri 是一个支持多协议的代理工具，强调快速连接与自动切换功能。它可以根据延迟自动选择最优节点，减少手动切换的繁琐过程，保证网络体验的流畅性。
VLESS + XTLS 在移动网络的优势？
XTLS 模式减少握手次数和加密开销，适合高延迟或不稳定的移动网络。结合策略组和多节点切换，可提升网页浏览、视频播放和游戏的连接稳定性和流畅度。
VLESS 相比 VMess 的主要改进是什么？
VLESS 是 VMess 的后继协议，移除了加密层以减少服务器资源消耗，同时提供更灵活的加密与传输方式选择。它更轻量、可扩展性更好，常被用于与 XTLS 等新传输协议搭配使用。
SSR 如何手动导入节点？
SSR 节点通常以“ssr://”链接形式提供。用户可在客户端中点击“添加节点”或“导入链接”，粘贴该字符串后保存即可。程序会自动解析服务器地址、端口、加密方式等参数。
如何使用 Clash 配置订阅链接？
在 Clash 中添加订阅链接时，用户需复制提供的 URL 并在客户端的配置界面中粘贴。Clash 会自动下载并更新配置文件。通过订阅，用户可以一键更新节点信息，无需手动导入，适合长期维护多个节点来源的用户。
SSR 的加密方式推荐使用哪种？
SSR 支持多种加密方式，如 aes-256-cfb、chacha20、rc4-md5 等。建议优先选择 chacha20 系列算法，在移动设备上性能更佳，同时具有较强的安全性。
PassWall2 自动切换节点机制如何工作？
PassWall2 会根据节点延迟和丢包率测试结果，结合策略组自动选择最优节点处理流量。无需用户手动切换，保证网页、视频和游戏流量的速度和稳定性，优化网络体验。
Netch 的透明代理适合哪些应用场景？
透明代理通过 TUN 或 WinDivert 拦截系统 IP 层流量，无需配置每个应用。适合全局科学上网、在线游戏或对局域网设备统一代理的场景。
